## Projects

### QAPlibria
**Location**: [qaplibria/](qaplibria/)

Comprehensive Python library for solving Quadratic Assignment Problems (QAP).

- **Status**: Production ready
- **Language**: Python 3.8+
- **Features**: Multiple solvers, benchmarking, CLI
- **Docs**: [qaplibria/docs/README.md](qaplibria/docs/README.md)

**Quick Start**:
```bash
cd qaplibria
pip install -e .
qapctl --help
```
