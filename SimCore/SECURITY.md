# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to `security@alawein.com` with full reproduction steps and, if possible, a PoC or test case.

## Response Targets

- Initial acknowledgement: 48 hours
- Status update: 7 days

## Scope

This policy covers the active SimCore repository and its published deployments. Archived or companion repositories may follow different schedules.

## Security Best Practices

- Keep dependencies and toolchains up to date.
- Follow secure coding guidelines for React, Vite, and WebGPU interactions.
- Sanitize user-generated content before rendering or executing.
- Report suspected vulnerabilities privately before posting publicly.
