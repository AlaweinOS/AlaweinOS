# SimCore

Core simulation engine and utilities.

## Overview

SimCore is a foundational simulation engine providing core functionality for scientific computing and simulation projects.

## Installation

```bash
pip install -e .
```

## Quick Start

```python
from simcore import Simulation
sim = Simulation()
results = sim.run()
```
